This game is truly simple. You just need to guess the right color for the RGB code shown in the header.

If you did not managed to guess the right color, the clicked area fade and its RGB code will be written out. However, if you guessed it right, each box will be painted in the desired color.

If you are not fimiliar with the color codes do not worry, click the EASY button and and you can now choose from fewer samples.

The RGB stand for Red, Green and Blue respectively. The numbers go from 0 to 255. 0 means that there are no color at all, 255 means that the color is 100% pure.

Have fun and good luck!
